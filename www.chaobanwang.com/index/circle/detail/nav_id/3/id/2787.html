智友创业空间楼盘介绍

智友趣园由智友实业（上海）有限公司打造，园区位于上海市青浦区，占地面积30亩，办公楼、仓库、公寓达20000余平方米。紧邻大虹桥商务区，开车10多分钟可达虹桥机场。园区南靠 318国道，北近312国道和沪宁高速公路，是上海市连接外省市的必经之路，区位优势显著。

智友致力于“助力中小企业成为专精特新企业”，借鉴游乐场似的谷歌办公环境和硅谷等世界优质企业做法，赋予园区休闲、娱乐属性，“关门工作，开门生活”，在这里工作，可以随时休整，在艺术性的环境中放松或碰撞，心情愉悦，构建人、工作、工作环境的和谐关系。

园区现已打造完成国内办公楼内第一的热带雨林生态缸造景，命名为 “Re-Earth”，整个造景高13米、长8米、宽4米，采用滴灌、无土栽培、喷淋、太阳能追光镜、太阳能发电、空气能加热、废水收集处理过滤等一系列的生态造景技术，结合LED屏幕、投影灯、背景灯光及玻璃立体墙绘、生态景观咖啡吧，打造了一个沉浸式雨林生态空间。
